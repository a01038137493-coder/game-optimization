# GameBoost Pro — 윈도우 최적화 프로그램 기획서

---

## 1. 개요

| 항목 | 내용 |
|---|---|
| 제품명 | GameBoost Pro Optimizer |
| 플랫폼 | Windows 10 / 11 (64bit) |
| 기술 스택 | Electron + React (또는 WPF / Tauri 고려) |
| 배포 방식 | 설치형 `.exe` 또는 포터블 `.exe` |
| 주요 목적 | 게이머 PC 성능 최적화 자동화 + 전문가 원격 연동 |

---

## 2. 디자인 컨셉

### 2.1 전체 방향
- **다크 테마 전용** (게이머 감성, 눈 피로 최소화)
- 웹사이트 (`gameboostpro.co.kr`) 와 동일한 색상 시스템 유지 → 브랜드 일관성
- 미니멀하고 깔끔한 UI, 복잡한 설정 최소화
- 초보자도 버튼 하나로 사용 가능한 구조

### 2.2 색상 시스템
| 용도 | 색상 |
|---|---|
| 배경 (메인) | `#04080f` |
| 배경 (카드) | `#0d1f2d` |
| Primary (강조) | `#00e87a` (네온 그린) |
| Secondary | `#00b4ff` (사이버 블루) |
| Accent (경고/위험) | `#ff3c6e` |
| 텍스트 | `#e8f4ff` |

### 2.4 UI 컴포넌트 스타일
- **버튼**: 라운드 코너 (8px), 그라디언트 (primary → secondary)
- **카드**: 반투명 배경 + 네온 보더 (hover 시 글로우 효과)
- **프로그레스바**: 애니메이션 그라디언트 채움
- **토글 스위치**: 커스텀 초록 토글
- **아이콘**: Lucide 또는 Phosphor 아이콘셋
- **폰트**: Pretendard (한글) / Inter (영문)

## 3. 기능 목록

### 3.1 🏠 홈 (대시보드)
- **시스템 현황 요약 카드**
  - CPU 사용률 (실시간 게이지)
  - RAM 사용률 / 전체 용량
  - 디스크 상태 (C드라이브 여유 공간)
  - 현재 온도 (CPU / GPU)
- **마지막 최적화 날짜** 표시
- **빠른 최적화 버튼** (원클릭 전체 최적화)
- **PC 점수** (0~100점, 최적화 전후 비교)

---

### 3.2 ⚡ 최적화

#### A. 원클릭 최적화
- 모든 항목 자동 분석 후 안전한 항목만 선택 적용
- 적용 전/후 성능 수치 비교 표시

#### B. 항목별 세부 최적화

| 카테고리 | 세부 기능 |
|---|---|
| **메모리** | 불필요한 메모리 해제, 가상 메모리 최적화 |
| **프로세스** | 백그라운드 불필요 프로세스 종료 목록 제시 |
| **시작프로그램** | 시작 프로그램 관리 (활성화/비활성화) |
| **전원 설정** | 고성능 전원 계획 적용, Ultimate Performance 활성화 |
| **디스크** | 임시 파일 삭제, 휴지통 비우기, 디스크 정리 |
| **네트워크** | DNS 캐시 플러시, TCP/IP 최적화, 네트워크 우선순위 조정 |
| **레지스트리** | 키보드 입력 지연 최적화, 게임 모드 레지스트리 적용 |
| **서비스** | 불필요 Windows 서비스 비활성화 (안전 목록만) |
| **GPU** | 하드웨어 가속 설정, 그래픽 우선순위 게임으로 조정 |

#### C. 게임 모드 진입
- 최적화 실행 전 "게임 모드 ON" 버튼
- 실행 중인 불필요 프로세스 일괄 종료
- 완료 후 "게임 모드 OFF" 복원 가능

---

### 3.3 🎮 게임별 최적화 프로파일

각 게임에 맞는 최적화 설정을 프리셋으로 제공

| 게임 | 주요 최적화 포인트 |
|---|---|
| 발로란트 | 입력 지연 최소화, 렌더링 스레드 최적화 |
| 리그오브레전드 | CPU 우선순위 조정, 네트워크 버퍼 최적화 |
| 배틀그라운드 | GPU 메모리 설정, 셰이더 캐시 관리 |
| 오버워치2 | 전원 성능 최대화, 멀티스레드 최적화 |
| 에이팩스 레전드 | 프레임 제한 해제, 입력 레이턴시 감소 |
| 타르코프 | RAM 최적화, 가상 메모리 확장 |
| 러스트 | 서버 연결 최적화, 물리 엔진 스레드 조정 |
| 아크: 서바이벌 | 디스크 I/O 최적화, 텍스처 캐시 관리 |

### 3.4 🔍 진단 (PC 건강 체크)

- **전체 진단 스캔** (소요 시간: 약 30초~1분)
- 진단 항목:
  - 드라이버 업데이트 필요 여부
  - 디스크 오류 검사 (S.M.A.R.T.)
  - 악성 시작 프로그램 감지
  - 고온 경고 (CPU/GPU 임계 온도 초과)
  - 메모리 누수 프로세스 탐지
  - 불필요 대용량 파일 목록
- **진단 리포트** 저장 (PDF 또는 텍스트)

---

### 3.5 📊 실시간 모니터링

- FPS 오버레이 (게임 실행 중 화면 위 표시, 위치 조정 가능)
- CPU / GPU / RAM 실시간 그래프
- 온도 모니터링 + 경고 알림
- 네트워크 지연(핑) 실시간 표시

---

### 3.6 ⚙️ 설정

- 앱 시작 시 자동 실행 여부
- 최적화 예약 (매일/매주 특정 시간에 자동 실행)
- 복원 지점 생성 (최적화 전 레지스트리 백업)
- 언어 설정 (한국어 / 영어)
- 업데이트 자동 확인

## 4. 프리미엄 기능 (유료 연동)

| 기능 | 설명 |
|---|---|
| **전문가 원격 연결** | 앱 내에서 GameBoost Pro 전문가 원격 서비스 바로 신청 |
| **고급 레지스트리 최적화** | 자체 개발 심층 레지스트리 튜닝 (일반 버전 미포함) |
| **자동 업데이트 드라이버** | GPU/네트워크 드라이버 자동 최신화 |
| **클라우드 설정 백업** | 최적화 설정을 계정에 저장, 다른 PC에서 복원 |

---

## 5. 안전 정책

- 레지스트리 수정 전 **자동 백업** 필수
- 시스템 핵심 프로세스 (svchost, winlogon 등) 절대 종료 불가
- 각 최적화 항목에 **"되돌리기"** 버튼 제공
- 안전 등급 표시: 🟢 안전 / 🟡 주의 / 🔴 고급 사용자용

---

## 6. 기술 스택 후보

| 옵션 | 장점 | 단점 |
|---|---|---|
| **Electron + React** | 웹 기술 재사용, UI 자유도 높음 | 용량 큼 (100MB+) |
| **Tauri + React** | 용량 작음 (10MB대), 빠름 | Rust 백엔드 필요 |
| **WPF (.NET)** | 윈도우 API 접근 용이, 성능 좋음 | 디자인 구현 어려움 |
| **PyQt / PySide** | 개발 빠름 | 배포 복잡, UI 한계 |

> **추천**: Tauri (Rust + React) — 가볍고 빠르며 웹사이트 UI 재활용 가능
