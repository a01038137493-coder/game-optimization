    FAQ

    자주 묻는 질문

    궁금한 점이 해결되지 않으면 카카오 채널로 문의해주세요.

    AnyDesk · TeamViewer 등 공인된 원격 프로그램만 사용합니다. 작업 중 파일 열람·복사·전송은 절대 하지 않으며, 작업이 끝나면 즉시 연결을 종료합니다. 고객님이 화면을 직접 보면서 진행되기 때문에 모든 과정을 실시간으로 확인할 수 있습니다.
    서비스별로 다르지만 대부분 30~90분 내에 완료됩니다. Lite 포맷은 약 40~60분, Windows 최적화는 30~50분, 포맷+최적화는 60~90분, 듀얼부팅은 90분 내외입니다. 작업 전 예상 소요시간을 안내드립니다.
    포맷 서비스는 윈도우를 새로 설치하는 과정이므로 C드라이브(윈도우 설치 드라이브)의 데이터는 삭제됩니다. 작업 전 반드시 중요한 파일을 백업해 두시거나, 별도 드라이브(D드라이브 등)에 저장해 두시면 안전합니다. Windows 최적화 서비스는 포맷 없이 진행되어 데이터가 유지됩니다.
    작업 완료 후 7일 이내 작업과 관련된 문제가 발생하면 무료로 재작업을 진행합니다. 카카오 채널로 문의주시면 빠르게 대응해드립니다.
    주로 AnyDesk를 사용합니다. 고객님이 편한 프로그램이 있다면 그걸로도 가능합니다. 설치 방법을 모르셔도 카카오 채널에서 단계별로 안내해드리니 걱정 안 하셔도 됩니다.
